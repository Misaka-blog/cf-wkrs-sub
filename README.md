## 注意事项

1. 由于默认 workers 从节点池的海量节点筛选时，不是按照国内环境筛选，因此需要二次在本地测试所有节点，筛选后使用可用的节点
2. 由于 workers.dev 域名被墙，所以需要绑定自己的域名
3. 由于节点均为公用节点，因此请勿在节点上登录自己的重要账户数据
4. 请勿滥用！
